# Вимоги

## Бізнес-можливості
- реєстрація й автентифікація користувачів
- ролі `user`, `moderator`, `admin`
- створення оголошень
- категоризація оголошень
- модерація перед публікацією
- повідомлення між користувачами
- пошук, фільтрація та сортування каталогу

## Нефункціональні вимоги
- чистий код і розділення на шари
- REST API через FastAPI
- безпечна JWT-автентифікація
- тести unit + integration
- контейнеризація
- CI/CD

## Технічний стек
- Python
- FastAPI
- MongoDB
- Docker
- GitHub Actions
