# Тестування

## Підхід
- unit тести перевіряють сервіси
- integration тести перевіряють REST API
- bootstrap smoke тести перевіряють Mongo-підхід та numeric counters

## Інструменти
- `pytest`
- `httpx`
- `mongomock`

## Поточний фокус
- реєстрація та логін
- CRUD категорій
- життєвий цикл оголошення
- модерація
- повідомлення
- права доступу
